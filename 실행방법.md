# 🚀 Study Helper 실행 방법

> **LangGraph 기반 멀티에이전트 학습 도우미** — 백엔드(FastAPI) + 프론트엔드(React/Vite) 구성

---

## ✅ 사전 준비

| 항목 | 요구 버전 |
|---|---|
| Python | 3.10 이상 권장 |
| Node.js | 18 이상 권장 |
| npm | Node.js 설치 시 함께 제공 |
| OpenAI API Key | 필수 |

---

## 📋 전체 실행 순서 요약

```
1. Python 패키지 설치
2. .env 파일 설정 (API 키)
3. 강의자료 PDF 인덱싱 (최초 1회)
4. 백엔드 서버 실행       ← 터미널 1
5. 프론트엔드 서버 실행   ← 터미널 2 (별도)
```

---

## 1️⃣ Python 패키지 설치

프로젝트 루트 디렉터리(`study_helper_web/`)에서 실행합니다.

```bash
pip install -r requirements.txt
```

> 처음 설치 시 `sentence-transformers`, `chromadb` 등의 패키지로 인해 시간이 다소 걸릴 수 있습니다.

---

## 2️⃣ 환경변수(.env) 설정

프로젝트 루트에 `.env` 파일이 이미 존재합니다.  
**OpenAI API Key** 를 본인 키로 교체하여 저장하세요.

```env
OPENAI_API_KEY="sk-xxxxxx..."

# (선택) LangSmith 트레이싱 — 없어도 실행 가능
LANGCHAIN_TRACING_V2=true
LANGCHAIN_ENDPOINT="https://api.smith.langchain.com"
LANGCHAIN_API_KEY="lsv2_pt_xxxxxx..."
LANGCHAIN_PROJECT="study_helper_v2"
```

> ⚠️ LangSmith 설정이 없는 경우 `LANGCHAIN_TRACING_V2=false` 로 바꾸거나 해당 줄을 삭제하세요.

---

## 3️⃣ 강의자료 인덱싱 (최초 1회 필수)

강의자료 PDF 파일을 준비합니다.

```bash
# 1. data/ 폴더에 PDF 파일을 복사합니다.
#    파일명은 반드시 rag.pdf 로 지정해야 합니다.
copy 강의자료.pdf data\rag.pdf

# 2. 인덱싱 스크립트 실행 (ChromaDB + BM25 인덱스 생성)
python scripts/ingest.py
```

실행 후 `chroma_db/` 폴더와 `bm25_index.pkl` 파일이 생성되면 성공입니다.

> ✅ 이미 `chroma_db/`와 `bm25_index.pkl`이 존재하면 이 단계를 건너뛸 수 있습니다.

---

## 4️⃣ 백엔드 서버 실행 (터미널 1)

```bash
uvicorn api:app --reload
```

- 서버 주소: **http://localhost:8000**
- `--reload` 옵션은 코드 변경 시 서버를 자동 재시작합니다.
- API 문서(Swagger UI): http://localhost:8000/docs

---

## 5️⃣ 프론트엔드 서버 실행 (터미널 2 — 별도 터미널)

```bash
cd frontend
npm install      # 최초 1회만 실행 (node_modules 생성)
npm run dev
```

- 웹 앱 주소: **http://localhost:5173**
- 백엔드 서버가 먼저 실행 중이어야 합니다.

> ✅ `node_modules/` 폴더가 이미 존재하면 `npm install`은 생략 가능합니다.

---

## 6️⃣ (선택) CLI 모드 실행

웹 브라우저 없이 터미널에서만 사용하려면:

```bash
python main.py
```

터미널에서 개념을 입력 → 문제 생성 → 답안 입력 → 채점 순서로 진행됩니다.

## 🛠️ 자주 발생하는 오류 해결

### ❌ `ModuleNotFoundError` 발생 시
```bash
pip install -r requirements.txt
```
패키지가 누락된 경우 위 명령으로 재설치합니다.

### ❌ `OPENAI_API_KEY` 관련 오류 시
`.env` 파일의 API 키가 올바른지 확인하고, 파일이 프로젝트 루트(`study_helper_web/`)에 있는지 확인합니다.

### ❌ 포트 충돌 오류 시
다른 포트를 지정하여 실행합니다.
```bash
# 백엔드를 8001 포트로 실행
uvicorn api:app --reload --port 8001

# 프론트엔드를 다른 포트로 실행
npm run dev -- --port 5174
```

### ❌ `chroma_db` 관련 오류 시
강의자료 인덱싱(3단계)이 완료되지 않은 경우입니다. `scripts/ingest.py`를 다시 실행하세요.
